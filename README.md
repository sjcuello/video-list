## How to install it?
___

```
git clone https://github.com/sjcuello/video-list.git
cd video-list
yarn install
```

## How to run it ?
___

+ Excecute the following command from `~/video-list`:
```
yarn start
```
+ Open [http://localhost:8080](http://localhost:8080) to view it in the browser.
